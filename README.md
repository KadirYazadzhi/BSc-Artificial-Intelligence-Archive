## 🛠️ Completed Projects

### 🔵 Semester 1: Basic Programming
**Project:** [Real Estate Agency Information System](./Semester-1-Basic-Programming/)

This project was the foundational entry into software development. It focused on mastering C++ syntax, modular code structure, and persistent data management.

*   **Core Concepts:** Arrays, Structs, C-style and C++ File I/O, Modular Programming.
*   **Key Features:**
    *   **Data Persistence:** Multi-layered strategy with automatic recovery (`sync.txt`/`sync.dat`) and manual binary backups.
    *   **Localization:** A custom key-value translation system for English and Bulgarian.
    *   **Hybrid Sorting:** Implementation of Quicksort and Merge Sort for property management.
    *   **Robustness:** Extensive input validation and string sanitization.

### 🟢 Semester 2: Data Structures and Algorithms (DSA)
**Project:** [Real Estate Agency - DSA Edition](./Semester-2-Data-Structures-Algorithms/)

The second semester involved a complete refactoring of the previous system to implement advanced data structures and evaluate algorithmic complexity.

*   **Core Concepts:** Dynamic Memory Management, Linked Lists, Recursion, Time/Space Complexity.
*   **Key Features:**
    *   **Dynamic Linked List:** Replaced static arrays with a dynamic `Node`-based list for flexible property management.
    *   **Recursive Search:** Implementation of a recursive binary search algorithm.
    *   **Algorithm Suite:** Comparative implementation of multiple sorting algorithms including **Bubble Sort** (on linked lists), **Selection Sort**, **Quicksort**, and **Merge Sort**.
    *   **Academic Rigor:** Focus on memory safety and efficient data traversal.
